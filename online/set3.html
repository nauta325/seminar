영상에서 만드는 것들

출결 핀(PIN) 등·하원 + 현황판하이라이트

학생이 키오스크에서 핀만 누르면 등·하원 자동 기록. 웹앱 현황판으로 실시간 확인, 결석은 자동 기록.

등·하원 카카오 알림톡 자동 발송

"○○ 등원했습니다" 알림톡이 학부모에게 자동 전송. 앱스크립트가 솔라피를 직접 호출합니다.

데일리리포트 자동 생성

오늘 올 학생을 코드가 알아서 판단 → 등원생별 리포트 빈 양식 자동 생성. 선생님은 진도·태도만 채우면 끝.

월말평가서 AI 자동 작성하이라이트

한 달 누적 수업 기록을 AI가 모아 평가서 본문까지 자동 작성. 며칠 걸리던 일이 30분으로.

학부모 열람 페이지 + 자동 발송

출석률·태도·진도가 정리된 학부모 전용 페이지가 자동 생성 → 링크 발송까지.

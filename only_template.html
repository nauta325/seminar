PC 브라우저 권장

  1. 위 링크 클릭
    PC 브라우저를 권장합니다. 모바일에서도 가능하지만 복제 버튼이 메뉴 안에 숨어있을 수 있어요.
  2. 우상단 "복제(Duplicate)" 버튼 클릭
    페이지가 열리면 오른쪽 위에 복제 버튼이 보입니다.
  3. 워크스페이스 선택
    여러 개면 학원용 워크스페이스를 선택하세요. 하나만 있으면 자동으로 거기에 복사됩니다.
  4. 완료! 사이드바 확인
    내 노션 사이드바에 페이지가 새로 생깁니다. 클릭하면 4개 DB가 관계형까지 살아있는 상태로 들어와 있어요.

FAQ

자주 막히는 부분

"복제" 버튼이 안 보여요
모바일이면 우상단 ··· 메뉴 안에 숨어있을 수 있어요. PC 브라우저로 다시 접속하시면 잘 보입니다.
어느 워크스페이스에 복사해야 해요?
본인 학원용 워크스페이스에 복사하는 걸 추천합니다. 개인용이랑 분리해두면 나중에 관리가 편해요.
복제했는데 페이지가 안 보여요
브라우저 새로고침(Ctrl+R / Cmd+R) 한 번. 또는 사이드바 맨 아래 "개인 페이지" 섹션을 펼쳐보세요.
롤업(전화번호·교재비합계)이 빈값으로 떠요
복제 직후엔 정상입니다. 학생 데이터를 입력하고 학생이름 관계형 연결이 되면 자동으로 채워져요. 데이터 입력 후에도 빈값이면 학생이름 관계가 비어있는지 확인.
기본 세트랑 학원관리 허브, 둘 다 복제해야 하나요?
먼저 ① 기본 세트(4개 DB)를 복제해서 익숙해진 뒤, 한 화면에서 운영하고 싶어지면 ② 허브를 복제하세요. 허브만 따로 써도 되지만, 기본 세트부터 시작하는 걸 추천해요.
복제 후 Make 자동화도 같이 따라오나요?
❌ Make 시나리오는 따라오지 않습니다. Make는 별도로 본인 계정에서 새로 만드셔야 해요. (세미나 Chapter 4에서 다룸)

천천히 시간 두고 복제해서 본인 학원에 맞게 세팅해보세요.
막히는 부분은 언제든 카톡으로 문의주세요.

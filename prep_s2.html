STEP 3 — 카카오톡 알림톡 준비
STEP 05
카카오톡 비즈니스 채널 만들기
center-pf.kakao.com — 학원 채널 개설 (즉시 가능)
완료 ✓
  • 브라우저에서 center-pf.kakao.com 접속
  • 평소 쓰는 카카오 계정으로 로그인 (개인 카톡 계정 OK)
  • 오른쪽 위 "채널 만들기" 클릭
  • 채널 이름, 카테고리(교육 → 학원/공부방) 선택
  • "검색용 ID" 정하기 — 영문 소문자/숫자 가능
  • 채널 만들기 완료!
💡 꿀팁 — 검색용 ID
이 ID가 나중에 솔라피 연동할 때 꼭 필요해요. 학원 영문이름으로 깔끔하게 만들어두세요. 한 번 정하면 바꾸기 까다로워요.
⚠️ 주의
카카오톡 채널은 사업자등록증이 없어도 개인 카카오 계정으로 개설 가능해요. 단, 알림톡 발송 인증 단계에서 사업자 정보가 필요할 수 있습니다.
👉 카카오톡 채널 관리자 센터
STEP 06
솔라피에서 카카오 알림톡 신청
방금 만든 채널을 솔라피에 연동해요
완료 ✓
  • 솔라피 로그인 → 왼쪽 메뉴 "카카오 알림톡" 클릭
  • "발신 프로필 등록" 또는 "채널 추가" 버튼 클릭
  • STEP 5에서 만든 검색용 ID 입력 → 채널 정보 확인
  • 카카오톡으로 본인 인증 메시지가 와요 (관리자 폰)
  • 카톡에서 인증 완료하면 — 솔라피에 채널이 연결됩니다!
💡 참고
STEP 5의 채널이 있어야만 이 단계가 가능해요. STEP 5 먼저 완료해주세요.
STEP 07
알림톡 템플릿 4개 미리 신청해두기
⚠️ 승인에 1~2영업일! 세미나 2의 핵심 준비물
완료 ✓
  • 알림톡은 카카오톡으로 학부모에게 안내문을 보내는 거예요
  • 세미나 2에서는 앱스크립트가 솔라피를 직접 호출해 알림톡을 보내요 (Make 없이)
  • 발송하려면 "템플릿"을 미리 등록하고 카카오 심사를 받아야 해요 (심사 1~2영업일)
  • 세미나 2에서는 아래 4개 템플릿을 씁니다. 미리 신청해두세요 👇
📋 신청할 템플릿 4개

① 등원 알림 → 학생이 등원하면 학부모에게 자동 발송
변수: 학생이름·등원시간·하원시간

② 하원 알림 → 하원 시 자동 발송
변수: 학생이름·하원시간

③ 미등원 알림 → 예정 시간에 안 오면 발송
변수: 학생이름·예정시간

④ 월말평가서 → 월말평가서 열람 링크 발송
변수: 학생이름·연월·링크
💡 문구는 나중에 바꿔도 돼요
지금은 "이 4가지 용도의 템플릿이 있다"는 것만 알고 신청해두면 돼요. 정확한 문구 양식은 수강 확정 시 안내해 드리고, 학원에 맞게 바꾸는 법은 세미나에서 알려드려요.
⏰ 가장 중요한 준비물!
세미나 2는 알림톡 발송까지 직접 실습해요. 심사가 1~2영업일 걸리니 반드시 세미나 전에 4개 모두 미리 신청해두세요.

구글(Google) 계정 로그인 확인

STEP 09
노션 DB 템플릿 복제해두기
제공된 템플릿을 내 노션으로 복사 — 직접 안 만들어도 돼요
완료 ✓
  • 세미나 2는 노션 DB 6개가 필요해요 (학생·데일리리포트·교재·결석보강·월말참고사항 등)
  • 직접 만들면 속성명이 틀리기 쉬워서 — 완성된 템플릿을 드려요
  • 아래 링크 접속 → 우측 상단 "복제(Duplicate)" 클릭 → 내 노션 워크스페이스로 복사
  • 복사되면 끝! 속성은 이미 다 맞춰져 있어요. 학생 데이터만 채우면 돼요
💡 세미나 1 들으신 분
전체학생·데일리리포트 DB는 이미 있으실 거예요. 그래도 템플릿을 복제해두면 교재·결석보강·월말참고사항 DB까지 한 번에 맞춰져서 편해요.

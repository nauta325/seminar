STEP 1 — GitHub (내 원서 앱 저장소)
STEP 01
GitHub(깃허브) 회원가입·로그인
이 계정으로 원서 앱 코드를 내 것으로 복제해요
완료 ✓
  • 브라우저에서 github.com 접속 → 우측 위 Sign up (이미 계정이 있으면 로그인)
  • 이메일·비밀번호·사용자 이름(username) 입력해서 가입
  • 노트북 크롬에서 로그인이 되는지 확인해요
💡 GitHub가 뭐예요?
코드를 보관하는 곳이에요. 세미나에서는 "Use this template" 버튼으로 완성된 원서 앱을 내 저장소로 복제해서 시작해요. 코딩을 몰라도 계정만 있으면 됩니다.
👉 GitHub 바로가기
STEP 02
GitHub 이메일 인증 완료
인증을 안 하면 저장소 복제가 막힐 수 있어요
완료 ✓
  • 가입하면 GitHub가 인증 메일을 보내요
  • 메일함에서 Verify(인증) 링크를 클릭해요
  • 스팸함도 확인해보세요
💡 참고
예전부터 쓰던 GitHub 계정이면 이미 인증돼 있어요. 이 단계는 넘어가도 됩니다.
🗄️
STEP 2 — Supabase (데이터베이스)
STEP 03
Supabase(수파베이스) 회원가입
우리 앱의 데이터가 저장되는 무료 클라우드 데이터베이스예요
완료 ✓
  • 브라우저에서 supabase.com 접속 → 우측 위 Start your project(또는 Sign in) 클릭
  • ⭐ "Continue with GitHub"로 로그인하면 제일 편해요 (STEP 1에서 만든 계정)
    — 이메일로 가입하면 인증 메일의 링크를 눌러 인증까지 해두세요
  • 로그인되면 대시보드(Dashboard) 화면이 열려요 — 준비는 여기까지면 OK!
  • 결제 카드 필요 없어요 — 무료 요금제(Free)로 세미나 실습에 충분합니다
💡 Supabase가 뭐예요?
내 원서 앱의 데이터를 보관하는 무료 클라우드 데이터베이스예요. 우리가 만든 반·학생·등록한 책·퀴즈·학습 결과·녹음 정보가 전부 여기에 저장돼요. 인터넷에 있는 엑셀 표 창고라고 생각하면 쉬워요. 앱 화면에서 학생을 추가하면 그 정보가 Supabase에 쌓이고, 다시 열 때 여기서 불러오는 거예요.
🛠 수업 때 함께 할 일 (지금은 안 해도 됨)
① 새 프로젝트 만들기 → ② 준비된 SQL을 한 번 실행(표·저장공간이 자동으로 만들어짐) → ③ 주소(URL)와 API 키를 복사해서 Vercel에 넣기.
이 3단계는 세미나에서 화면 같이 보며 진행해요. 지금은 가입·로그인만 해두면 충분해요!
🔑 미리 알아두기
Supabase의 service_role 키는 아주 강력한 비밀키예요. 수업 때 Vercel 환경변수에만 넣고, 카톡·화면공유·HTML 코드에는 절대 붙여넣지 않아요. (이 부분은 수업에서 다시 안내해요.)
👉 Supabase 바로가기

STEP 3 — Vercel (배포·인터넷에 올리기)
STEP 04
Vercel(버셀) 회원가입 — GitHub로 연결
내 원서 앱을 인터넷 주소로 만들어주는 무료 서비스
완료 ✓
  • vercel.com 접속 → Sign Up
  • ⭐ "Continue with GitHub" 선택 — STEP 1 계정과 연결하면 배포가 자동으로 쉬워져요
  • 로그인되면 대시보드가 열려요 — 여기까지만!
💡 Vercel이 뭐예요?
내 저장소를 골라서 Deploy 버튼만 누르면 내이름.vercel.app 주소가 생기는 서비스예요. 수업 때 클릭 몇 번으로 배포해요.
👉 Vercel 바로가기

STEP 4 — Claude API (AI 자동생성)
STEP 05
Anthropic(Claude) 콘솔 가입·로그인
지문만 붙여넣으면 단어카드·퀴즈를 AI가 자동 제작
완료 ✓
  • console.anthropic.com 접속 → 가입/로그인
  • 구글 계정 또는 이메일로 가입할 수 있어요
💡 왜 필요해요?
원서 앱의 ✨AI 자동생성 기능(영어 지문 → 단어카드·원서퀴즈 2종 자동 제작)에 Claude를 써요. 이 기능을 쓰려면 API 키가 필요해요.
👉 Claude 콘솔 바로가기
STEP 06
결제수단 등록 + 소액 충전
⚠️ 카드 승인에 시간이 걸릴 수 있어요. 미리 해두세요!
완료 ✓
  • 콘솔 → Billing(결제) → 카드 등록
  • $5 정도만 소액 충전해두세요 (원서 몇 개 생성은 몇 백 원 수준이라 부담 적어요)
  • 충전 잔액이 있어야 API 키가 실제로 작동해요
💳 결제가 자꾸 실패한다면? — 크롬 언어를 영어로 바꾸세요!
한국어 브라우저에서 카드 등록·결제가 자꾸 오류(카드 거부·에러 메시지)가 나는 경우가 있어요. 이럴 땐 크롬 언어를 영어로 바꾸고 다시 시도하면 대부분 해결돼요:
  1. 크롬 우측 위 ⋮ → 설정(Settings) → 언어(Languages)
  2. English (United States) 를 추가하고 맨 위로 올리기
  3. "이 언어로 Chrome 표시" 켜고 크롬 다시 시작
  4. 결제 페이지 새로고침 후 카드 등록·충전 재시도
결제가 끝나면 언어는 다시 한국어로 돌려놔도 돼요 🙂
⏰ 미리 해두세요!
카드 등록·해외결제 승인에 시간이 걸릴 수 있어요. 당일 하면 실습이 늦어지니 세미나 전에 꼭 등록·충전해두세요.
STEP 07
API 키 발급 → 메모장에 저장
수업 때 Vercel 환경변수에 넣을 키예요
완료 ✓
  • 콘솔 좌측 API Keys → Create Key 클릭
  • 만들어진 키를 복사해서 메모장에 붙여넣어 저장 (창을 닫으면 다시 안 보여줘요!)
  • 이 키는 수업 때 Vercel 환경변수 ANTHROPIC_API_KEY에 넣어요
🔑 키는 비밀번호예요
API 키는 화면공유·채팅에 노출되지 않게 조심하고, 본인 메모장에만 저장해두세요. 노출되면 즉시 삭제하고 새로 발급하면 됩니다.

STEP 08
노트북 챙기기
맥북·윈도우 모두 OK. 충전기도 챙겨오세요
완료 ✓
  • 크롬 브라우저가 설치되어 있으면 좋아요
  • 충전기도 같이 챙겨오세요

스마트폰·태블릿만으로는 배포 실습이 어려워요. 반드시 노트북을 가져오세요!
STEP 09
스마트폰 챙기기
학생 화면의 "소리내어 읽고 녹음" 테스트용
완료 ✓
  • 학생 화면엔 소리내어 읽고 녹음하는 기능이 있어요
  • 폰으로 접속하면 녹음이 실제로 되는지 직접 확인할 수 있어요
  • 필수는 아니지만 있으면 더 재밌어요!
💡 꿀팁
폰 브라우저에서 앱을 "홈 화면에 추가"하면 진짜 앱처럼 실행돼요 (PWA).
STEP 10
테스트용 원서 지문 2~3개 준비
AI 단어카드·퀴즈 생성을 직접 실습해봐요
완료 ✓
  • AI 생성을 실습하려면 영어 지문이 필요해요
  • 우리 학원에서 쓰는 원서의 영어 텍스트 2~3개를 준비해오세요 (복사 가능한 형태)
  • 짧아도 됩니다 — 5~10문장이면 단어카드·퀴즈가 잘 나와요
💡 참고
저작권이 걱정되면 짧은 발췌나 직접 만든 예문도 좋아요. 준비 못 하셔도 수업 중에 예시 지문을 드리니 걱정 마세요.
STEP 11
학원 기본 정보 정리해 오기
앱을 우리 학원에 맞게 바꾸는 실습(바이브 코딩)에 써요
완료 ✓
  • 수업 때 앱을 우리 학원에 맞게 바꿔요. 아래를 미리 떠올려 오세요:
  • 학원 이름 / 원하는 대표 색 / 통과 점수(예: 70점)
  • 우리 학원 레벨 목록(예: AR 단계) / 자주 쓰는 원서 시리즈 목록
  • 테스트용 학생 이름 3~5명 (가명 OK)
💡 왜 미리?
이 항목들은 각 파일에서 한 줄만 바꾸면 되는 것들이라, 미리 정리해오면 수업이 훨씬 빨라져요.

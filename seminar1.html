02 — CURRICULUM · 당일 만드는 것들

수업이 끝나면
작동하는 시스템이 손에 남습니다

기능 설명만 듣고 끝나는 수업이 아닙니다. 따라 만들면, 그날 바로 내 학원에 쓸 수 있는 노션 시스템이 완성됩니다.

  • 기초
    노션 기초 · 데이터베이스 다루기
    페이지·블록 개념부터 표 만들기, 속성 추가, 그리고 같은 데이터를 표·보드·캘린더·갤러리로 바꿔 보는 뷰 전환까지. 필터·정렬·그룹화로 "재원생만", "요일별로", "선생님별로" 원하는 대로 꺼내봅니다.
    왜 필요한가
    학원 시스템의 모든 게 DB 위에 올라갑니다. 여기가 흔들리면 나머지가 다 흔들려요.
    끝나면
    노션 화면이 더는 막막하지 않고, 데이터를 원하는 모양으로 꺼낼 수 있게 됩니다.
  • 실습
    학생 DB + 교재 DB 구축
    학생 명단·연락처·재원상태, 사용 교재·유닛 목록을 데이터베이스로 정리합니다. 학원 운영의 뼈대가 되는 두 핵심 DB를 직접 만듭니다.
    만드는 것
    전체학생 DB(이름·연락처·학교·학년·재원상태·등원시간), 교재 DB(교재명·종류·가격·재고)
    끝나면
    학생·교재 정보가 한 곳에 모여 검색·필터로 즉시 찾을 수 있습니다.
  • 핵심 개념
    DB끼리 연동하기 — 관계형 + 롤업
    학생 DB와 교재 DB를 관계형으로 연결합니다. 수업 기록에서 학생 이름을 선택하면 연락처가 자동으로 따라오는 그 마법, 직접 만들어 봅니다. 롤업으로 교재비 합계·배포 수량도 자동 계산.
    왜 어렵게 느껴지나
    관계형·롤업은 노션의 진짜 힘인데 대부분 여기서 막힙니다. 차근차근 손으로 같이 만듭니다.
    끝나면
    한 번 입력한 정보가 여러 DB에서 자동으로 연결돼, 같은 걸 두 번 적을 일이 없어집니다.
  • 실습
    출결·보강 관리 + 요일별 시간표
    하나의 DB를 여러 뷰로 — 오늘 보강, 일주일 뒤 보강(캘린더 뷰), 이번 달 결석을 한눈에. 월~금 시간표는 그룹화로 요일별 블록 보기까지 만듭니다.
    실전 팁
    보강 캘린더의 "새 페이지" 문제, 날짜 색 안 되는 문제 등 실제 막히는 지점도 같이 해결합니다.
    끝나면
    누가 언제 보강인지, 이번 주 시간표가 어떤지 한 화면에서 봅니다.
  • 실습
    버튼 + 원장 스케줄 캘린더
    클릭 한 번으로 오늘 날짜 리포트·일정이 자동 추가되는 버튼을 만듭니다. 상담·행정·수업 일정을 분류별 색으로 보는 원장 전용 캘린더로 하루를 한눈에 관리합니다.
    만드는 것
    "오늘 리포트 추가" 버튼, 일정 분류(상담/행정/수업)별 색상 캘린더
    끝나면
    매일 반복하던 입력을 버튼 한 번으로, 하루 일정을 색으로 한눈에 봅니다.
  • 하이라이트
    일일 리포트 DB + 학부모 문자 자동 완성
    수업 기록을 입력하면 학부모 문자가 수식으로 자동 완성됩니다 — 복사해서 카톡에 붙여넣기만 하면 끝. 타이핑 없이 10초면 발송.
    핵심
    진도·태도·과제·단어테스트를 선택하면 학부모용 문구가 자동으로 조립됩니다.
    끝나면
    매일 밤 문자 쓰던 30분이 10초로 줄어듭니다.
  • 실습
    교재비 관리 DB + 학부모 안내 문자
    교재 구입 내역·미납 현황을 DB로 관리하고, 수식으로 교재비 안내 문자를 자동 완성하는 방법까지.
    만드는 것
    배부월·결제상태(미납/완납)·교재비 합계가 자동 계산되는 교재비 DB
    끝나면
    누가 얼마 미납인지 즉시 보이고, 안내 문자도 클릭으로 완성됩니다.
  • 자동화
    Make로 일일 리포트 자동 SMS 발송
    노션에 수업 기록을 입력하면 Make가 감지 → 솔라피로 학부모 문자가 자동 발송됩니다. 당일 본인 폰으로 직접 테스트해 봅니다.
    카카오 알림톡은 템플릿 검수에 1~2영업일이 걸려 당일 테스트가 어렵습니다. 세미나 1에서는 SMS 자동 발송을 경험하고, 세미나 2에서 알림톡 직접 발송으로 업그레이드합니다.

04 — FAQ

자주 묻는 질문

노션을 한 번도 안 써봤는데 따라갈 수 있나요?

네. 페이지·블록 개념부터 시작합니다. 신청자 절반이 "노션 완전 처음"이라 왕초보 페이스로 진행해요. 코딩은 0줄입니다.

Make가 뭔가요? 어려운가요?

노션·문자 발송 같은 서비스를 코드 없이 연결해 주는 자동화 도구예요. 화면에서 블록을 잇는 방식이라, 당일 따라 만들고 본인 폰으로 바로 테스트합니다.

준비물이 있나요?

노트북 2대를 권장합니다(줌 화면용 + 노션 실습용). 노션 계정은 무료로 만들면 되고, 문자 발송 테스트용 솔라피는 당일 안내해 드려요.

세미나 2(앱스크립트)와 뭐가 다른가요?

세미나 1은 코딩 없이 노션 + Make로 만드는 입문 단계예요. 세미나 2는 그 위에서 Apps Script로 출결·알림톡·월말평가서까지 완전 자동화합니다. 1차가 모든 세미나의 기초라 먼저 듣길 권합니다.

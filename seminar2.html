02 — CURRICULUM · 당일 만드는 것들

손으로 하던 일이
코드로 사라집니다

Make가 하던 일을 Apps Script가 직접 합니다. 더 빠르고, 더 많은 학생을 동시에, 그리고 카카오 알림톡 직접 발송까지.

  • 하이라이트
    출결 핀(PIN) 등·하원 + 실시간 현황판
    학생이 키오스크에서 핀만 누르면 등·하원이 자동 기록됩니다. 웹앱 현황판으로 오늘 누가 왔는지 실시간 확인, 결석은 출결보강 DB에 자동 기록됩니다.
    왜 코드인가
    여러 학생이 동시에 핀을 눌러도 순서대로 정확히 기록 — 노션 수동 입력으로는 불가능한 영역.
    끝나면
    출석부를 따로 적을 필요가 없고, 현황판만 보면 지금 학원 상황이 한눈에.
  • 자동화
    등·하원 카카오 알림톡 자동 발송
    "○○ 등원했습니다" 알림톡이 학부모에게 자동 전송됩니다. Make 없이 Apps Script가 솔라피를 직접 호출해요. 세미나 1의 SMS에서 알림톡으로 업그레이드되는 지점.
    1차와 다른 점
    1차는 SMS, 2차는 정식 알림톡. 발송 비용도 저렴하고 학부모 신뢰도 올라갑니다.
    끝나면
    등·하원 즉시 학부모에게 알림이 가, 따로 챙길 일이 없어집니다.
  • 자동화
    데일리리포트 자동 생성
    오늘 올 학생을 코드가 알아서 판단 → 등원생별 리포트 빈 양식을 자동 생성합니다. 선생님은 진도·태도만 채우면 끝.
    핵심
    "오늘 누구 리포트를 만들지" 고민을 코드가 대신합니다. 시간표 기반 자동 판단.
    끝나면
    빈 양식이 미리 깔려 있어, 수업 직후 바로 채워 발송.
  • 하이라이트
    월말평가서 AI 자동 작성
    한 달 누적 수업 기록을 AI가 모아 평가서 본문까지 자동으로 작성합니다. 전체 학생을 한 번에, 또는 한 명씩 생성할 수 있어요. 이 세미나의 진짜 핵심.
    왜 핵심인가
    월말마다 학생 수만큼 글을 쓰던 그 며칠이, 버튼 한 번으로 끝납니다.
    제공
    완성 코드 + 평가서 톤을 내 학원에 맞게 바꾸는 커스터마이즈 프롬프트.
  • 하이라이트
    학부모 열람 페이지 + 자동 발송
    출석률·태도·진도가 정리된 학부모 전용 페이지가 자동 생성되고, 링크가 학부모에게 발송됩니다. 학원이 따로 정리할 필요가 없습니다.
    만드는 것
    학생별 열람 페이지(로그인=학생이름+출결번호) + 자동 발송 흐름
    끝나면
    학부모가 언제든 자녀 현황을 직접 확인 — 상담 문의가 줄어듭니다.
⚠️ 수강 전, 꼭 미리 준비하세요
카카오 알림톡 템플릿은 승인까지 1~2영업일이 걸립니다. 세미나 당일 바로 발송 테스트를 하시려면 수강 전에 미리 알림톡 템플릿을 신청해두셔야 합니다. 신청 방법과 템플릿 양식은 수강 확정 시 안내해 드립니다.

04 — FAQ

자주 묻는 질문

코딩을 전혀 모르는데 따라갈 수 있나요?

네. 완성 코드를 자료실에서 제공합니다. 직접 짜는 게 아니라, 복사해서 내 학원 정보(학원명·연락처 등)만 바꿔 붙이는 방식이에요.

세미나 1을 안 들었어도 되나요?

세미나 2는 노션·알림톡이 기본 전제입니다. 1차를 먼저 듣거나 영상 강의를 보고 오시길 권합니다. 그래야 가장 매끄럽게 따라올 수 있어요.

알림톡 템플릿은 왜 미리 신청해야 하나요?

카카오 알림톡은 템플릿 승인에 1~2영업일이 걸려서, 세미나 당일 바로 만들면 테스트를 못 합니다. 수강 확정 시 신청 방법과 양식을 안내해 드려요.

줌과 오프라인 중 뭐가 나은가요?

내용은 같습니다. 오프라인(7/5, 일산)은 막히는 부분을 바로 옆에서 봐드릴 수 있어 설치 성공률이 높고, 줌(7/4)은 어디서든 참여할 수 있어요.

월말평가서 AI는 어떤 도구를 쓰나요?

Apps Script가 한 달 수업 기록을 모아 AI API로 평가서 본문을 생성합니다. 평가서 톤을 내 학원에 맞게 바꾸는 커스터마이즈 프롬프트도 함께 드려요.

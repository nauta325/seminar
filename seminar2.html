평소엔 수업 후 노션에 태도·과제·테스트만 클릭으로 기록. 월말에 버튼을 누르면 나머지는 시스템이 합니다.

1

05 — 출결 시스템

PIN 하나로
등·하원이 끝납니다

QR도, 앱 설치도 필요 없어요. 학생이 PIN 4자리만 누르면 등·하원이 기록되고, 학부모에게 알림톡이 즉시 발송됩니다.

  • 📲

    PIN 입력 → 알림톡 즉시 발송

    "홍블랑 학생이 3:05에 등원하였습니다. 예상 하원 시간은 5:05입니다." 등원·하원 모두 자동 발송.

  • ⏱️

    지각 자동 판단 + 하원시간 자동 계산

    예정 3:00 + 유예 10분 → 3:11부터 지각. 등원시간 + 수업시간으로 예상 하원시간까지 자동 계산해 알림톡에 포함. 유예시간은 학원에 맞게 변경 가능.

09 — FAQ

자주 묻는 질문

그냥 ChatGPT에 평가서 써달라고 하면 안 되나요?

AI한테 글 써달라는 건 누구나 합니다. 차이는 데이터 수집부터 문서 생성·차트 리포트·학부모 전달까지 원클릭으로 이어진다는 거예요. 매번 데이터 정리해 복붙하고, 결과를 복사해 카톡 보내는 그 과정이 통째로 사라집니다.

코딩을 전혀 모르는데 따라갈 수 있나요?

네. 완성 코드를 자료실에서 제공합니다. 직접 짜는 게 아니라, 코드를 붙여넣고 API 키만 바꾸면 돼요. 설치는 30분 정도 걸립니다.

설치하는 데 뭐가 필요한가요?

노션(무료), 솔라피(월 1~2만원, 알림톡용), Claude API(월 5천원 수준)면 됩니다. 서버 비용은 0원이에요 — 구글 앱스크립트가 무료로 서버 역할을 합니다.

세미나 1을 안 들었어도 되나요?

세미나 2는 노션·알림톡이 기본 전제입니다. 1차를 먼저 듣거나 영상 강의를 보고 오시길 권합니다.

알림톡 템플릿은 왜 미리 신청해야 하나요?

카카오 알림톡은 템플릿 승인에 1~2영업일이 걸려서, 당일 만들면 테스트를 못 합니다. 수강 확정 시 신청 방법과 양식(등원·하원·미등원 템플릿)을 안내해 드려요.

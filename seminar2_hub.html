DB 세트

🗂️ 출결 · 월말평가서 DB 템플릿

교재 PART 2에서 쓰는 Notion DB 구조예요. 복제해서 내 워크스페이스로 가져온 뒤, 코드의 DB ID만 내 것으로 바꾸면 바로 연결돼요.

출결·평가서 DB 템플릿 복제하기 복제 →

복제 방법

  1. 위 링크 클릭 → PC 브라우저 권장
  2. 우상단 "복제(Duplicate)" 버튼 클릭
  3. 내 학원용 워크스페이스 선택
  4. 사이드바에 DB가 생기면 완료 → 교재 보고 DB ID 확인
💡 복제 후 할 일 — 코드(PART 4)에서 DB ID·API 키를 내 값으로 바꿔야 연결돼요. Make 자동화나 실제 학생 데이터는 복제에 안 따라오니, 교재 순서대로 채워주세요.

질문 & 도움받기

1주 1:1

💬 카톡 질문방

설치하다 막히면 화면 캡처 + 어디서 막혔는지 한 줄만 남겨주세요. 세미나 후 1주일간 1:1 답변도 갑니다.

⚠️ 카톡 질문방 링크 (주소 넣기) 입장 →

→ 오픈카톡 링크로 교체하세요

알림톡이 안 나가요
솔라피 발신 프로필이 활성인지, 템플릿 심사가 통과됐는지 확인하세요. 심사는 영업일 1~3일 걸려요. Make 본문과 등록 템플릿이 한 글자라도 다르면 거부돼요.
"apiKey length must be 16" 에러가 떠요
솔라피 API Key와 Secret을 바꿔 넣은 경우가 대부분이에요. API Key는 16자 — 콘솔에서 다시 복사하고, 앞뒤 공백이 없는지 확인하세요.
평가서 문체를 바꾸고 싶어요
교재 PART 7 참고. 코드에서 getReportGuideline을 찾아 프롬프트(한글 명령문)만 고치면 돼요. 코드를 짤 줄 몰라도 됩니다.
Claude API 키는 어디서 발급받나요
Claude Console → API 키 → 키 생성. 발급 직후 딱 한 번만 전체가 보이니 바로 복사해두세요. 실습용 키는 각자 발급받아 각자 환경변수에 넣습니다.

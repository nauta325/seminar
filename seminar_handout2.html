CHAPTER 1
앱스크립트가 뭐예요?
1
구글이 주는 무료 코드 도구

구글 앱스크립트(Apps Script)는 구글 계정만 있으면 무료로 쓰는 코드 실행 도구예요. 따로 설치할 것 없이 브라우저에서 script.google.com 으로 바로 써요.

이 수업은 코드를 짜는 게 아니라, 완성 코드를 복사 → 붙여넣기 → 내 값만 교체하는 수업이에요.
2
Make와 뭐가 달라요?

1차의 Make는 노션과 솔라피를 이어주는 중간 다리(월 사용량 제한 있음)였어요.

앱스크립트는 코드가 노션·솔라피를 직접 호출해요. 중간 다리가 없으니 제한도 없고, 출결·웹앱처럼 더 정교한 것까지 가능해요.

전체학생DB
  • 학생이름(제목), 학생ID(텍스트), 재원상태(선택 재원/퇴원), 담당선생님(선택·띄어쓰기X), 수업시간(숫자·분), 학년(텍스트/선택)
  • 출결PIN(텍스트) — 키오스크에서 학생이 누를 번호
  • 전화번호(전화/텍스트/수식 다 인식)
  • 요일별 등원시간: 월~금-등원시간 (단일/다중선택 둘 다 인식)
  • 교재 Relation 3개 → 교재DB: 문법교재 / 리딩교재 / 단어교재 (안 쓰는 과목은 비움)
2
데일리리포트DB (가장 중요)
  • 키: 제목 · 학생이름(Relation→전체학생DB, 월말평가서 조회 키·필수) · 날짜(날짜)
  • 자동 채워짐: 카톡전송(선택), 태도/과제/지각(선택), 문법·리딩·단어교재(Relation, 학생DB에서 복사), 실제등원시간/실제하원시간(텍스트)
  • 수업 후 수기: 문법진도/리딩진도/단어진도(텍스트, 예 Unit 5), 단어테스트_결과(선택)
  • Select 옵션 미리 생성: 태도(상/중/하) · 과제(100%완료/상/중/하) · 지각(정시등원/지각/결석) · 카톡전송(미전송/전송/실패) · 단어테스트(통과/불통과)
⚠️ 학생ID 수식 불필요 — 월말평가서는 학생이름 Relation으로 조회해요. 데일리리포트DB에 학생ID 수식 속성은 안 만들어도 됩니다.
3
교재DB (교재비·재고 관리용)
  • 교재명(제목) — 코드가 읽는 유일한 값
  • 종류(선택 문법/리딩/단어), 가격(숫자), 누적교재수량(숫자), 배포된 수량(롤업), 재고(수식)
💡 교재DB는 진도용이 아니에요 — 코드는 제목(교재명)만 읽어요. 나머지 가격·재고는 학원 운영용입니다.
4
결석보강DB
  • 학생이름(Relation)
  • 결석 예정일 / 보강 예정일 (날짜)
  • 보강시간, 비고
  • 담당 선생님(롤업/선택·띄어쓰기)
5
월말참고사항DB
  • 학생(Relation→전체학생DB·필수)
  • 연월(이름에 연월/기간/월 포함)
  • 참고사항(이름에 참고/코멘트/메모 포함)
  • 중학생(체크박스)
6
월말평가서DB (자동 생성 — 직접 안 만듦)

이 DB는 createReportDatabase() 함수를 1회 실행하면 코드가 알아서 만들어요. (제목·학생·평가서본문·진도·열람링크·카톡전송 등 속성 자동 포함)

✅ 자동 생성되는 것 — 월말평가서DB · 교재DB ID 연결 · 학부모 링크 URL. 이 세 가지는 손대지 않아도 됩니다.

출결 + 데일리리포트 자동화하이라이트
1
출결 키오스크 (학생 PIN)

웹앱 주소 뒤에 ?page=attendance 를 붙이면 키오스크 화면. 태블릿·폰을 학원 입구에 두면 학생이 자기 핀(PIN)을 눌러 등·하원합니다.

  • 학생이 핀 입력 → 실제등원시간이 데일리리포트DB에 자동 기록
  • 하원 시 다시 핀 입력 → 실제하원시간 기록
2
출결현황판 (선생님용)

웹앱 주소 뒤 ?page=attendance_board. 오늘 누가 왔고 안 왔는지 실시간으로 보여줘요.

  • 터치로 하원 처리 · 미등원 알림 발송
  • 교사 필터로 담당 학생만 보기 (설치 시 선생님A/B를 실제 이름으로 교체)
3
데일리리포트 자동 생성

매일 새벽 트리거(createTodayStudentsPlannerRows)가 오늘 올 학생을 요일로 판단 → 등원생별 리포트 빈 행을 자동 생성. 교재(Relation)는 학생DB에서 자동 복사돼요.

🌙 새벽 트리거
→
오늘 등원생 판단
→
리포트 행 자동생성
→
교재 자동복사
자동 vs 수기 — 교재·출결시간은 자동, 진도(유닛 번호)와 단어테스트 결과는 수업 후 수기 입력.

💬 카카오 알림톡 템플릿 4개
⚠️
먼저 — 미리 등록 + 검수

알림톡은 사전 등록 + 카카오 검수(승인)가 필요해요. 검수는 보통 영업일 1~3일. 세미나 전에 미리 신청해두세요. 승인 후 받은 템플릿 ID를 Code.gs [설정]에 넣습니다.

❗ 변수명을 글자까지 똑같이 — #{학생이름} 같은 변수명이 템플릿과 코드에서 한 글자라도 다르면 치환이 안 됩니다.

1부
PART 1 · 기초
바이브 코딩 기초 — 말로 코드를 다루기
1
바이브 코딩이란?

코드를 한 줄씩 짜는 대신, 클로드에게 "이렇게 만들어 줘 / 이렇게 바꿔 줘"라고 말해서 프로그램을 만드는 방식이에요. 문법·오타 걱정 없이, 무엇을 원하는지만 말합니다.

2
핵심 루프
①말하기
→
②돌려보기
→
③확인
→
④고치기

한 번에 완벽하지 않아도 돼요. 돌려보고 → 고치고를 반복하면 됩니다.

3
준비물 — Claude Code + Claude 유료 구독 핵심

바이브 코딩 도구는 Claude Code예요. 쓰려면 Claude 유료 구독(claude.ai Pro 이상)이 필요합니다. 노트북에 설치·로그인까지 해두세요.

⚠️ 두 개의 "Claude"를 꼭 구분하세요
① Claude 구독 + Claude Code = 내가 코딩할 때 쓰는 도구.
② ANTHROPIC_API_KEY = 원서 앱이 퀴즈를 만들 때 쓰는 열쇠 (2부에서 Vercel에 넣음).
→ 완전히 별개예요. 둘 다 필요합니다.
4
잘 말하는 법 — 구체적으로·하나씩·결과로
✗ 애매
"예쁘게 만들어 줘."
✓ 구체적
"제목을 가운데로, 대표 색을 주황색으로, 버튼을 더 크고 둥글게 해줘."
모르면 물어봐도 돼요 — "이 화면 뭐가 문제야?"처럼 질문하면 클로드가 설명하고 방법을 제안합니다. 잘못 건드려도 "방금 바꾼 거 되돌려줘"로 되돌릴 수 있어요.
2부
PART 2 · 연습
원서 시스템으로 연습 — 받아서 돌려보고 · 수정

빈 화면보다 완성된 앱을 고치는 게 쉬워요. 받아서 → 먼저 돌려보고 → 뜯어보고 → 우리 학원에 맞게 수정합니다.

1
받아오기
  • 템플릿 저장소(자료실) → Use this template로 내 저장소 복제 (없으면 Fork)
  • 그 저장소를 Claude Code로 열기 — 이제 클로드가 이 코드를 다뤄요
2
먼저 돌려보기 — Supabase(데이터) + Vercel(배포)
  • Supabase: New project → SQL Editor에 sql/mini-setup.sql 붙여넣고 Run → Project URL·service_role 키 복사
  • Vercel: 내 저장소 Import → 환경변수 3개 입력 → Deploy → /mini/ 열기
환경변수값필수
SUPABASE_URLSupabase Project URL✅
SUPABASE_SERVICE_KEYSupabase service_role 키✅
ANTHROPIC_API_KEYClaude 콘솔 키 (앱 AI 생성용)선택
⚠️ service_role 키·API 키는 Vercel에만. 카톡·화면공유·HTML에 붙여넣지 마세요.
3
뜯어보기 — 클로드에게 물어보기
🗣️ 이렇게 말하기
"이 프로젝트가 어떤 앱인지, 파일이 각각 무슨 역할인지 쉽게 설명해줘. 나는 코드는 잘 몰라."
4
수정하기 — 말로 하나씩
바꾸고 싶은 것이렇게 말하기
학원 이름"리포트·워크시트 학원 이름을 '홍보은영어'로 바꿔줘"
대표 색"앱 전체 색을 주황색 계열로 바꿔줘"
통과 점수"통과 점수를 75점으로 바꿔줘"
레벨·시리즈"레벨을 1~4단계로, 시리즈를 우리 책들로 바꿔줘"
AI 출제"원서퀴즈2를 5문항으로, 3단계 이상은 영어로 내줘"
저장하면 자동 반영 — "저장하고 올려줘"라고 하면 커밋·푸시되고, Vercel이 자동 재배포해서 몇 분 뒤 내 주소에 보여요.
5
막힐 때
증상해결
데이터가 안 뜸Vercel 환경변수 오타 확인 → 재배포
저장·등록 안 됨Supabase SQL 다시 실행
AI 생성 안 됨ANTHROPIC_API_KEY·잔액 확인
화면이 깨졌다"방금 바꾼 것 되돌려줘"
모르겠다에러를 복사해 클로드에게 붙여넣고 물어보기

PART 3 · 창작
내 프로젝트 만들기 — 처음부터 끝까지

방법은 2부와 똑같아요 — 말하고, 돌려보고, 고친다. 다른 점은 시작이 "빈 화면"이라는 것뿐. 예시로 단어 플래시카드를 만들어 봅니다.

★
새 프로젝트 세팅 — 4개 도구를 새로

2부는 원서 템플릿을 복제해 이미 연결돼 있었지만, 내 프로젝트는 4개를 새로 만들어 잇습니다.

예시
단어 플래시카드 — 이렇게 말해요
🗣️ ① 첫 요청
"영어 단어 플래시카드 웹페이지를 만들어줘. 단어(영어·뜻)를 넣으면 카드가 한 장씩 뜨고, 누르면 뒤집혀 뜻이 나와. 다음 버튼으로 넘기고, 폰에서도 크게. 파일 하나로."
🗣️ ② 돌려보기
"브라우저에서 열어서 보여줘."
🗣️ ③ 고치기
"'섞기' 버튼을 넣고, '알아요/모르겠어요' 중 모르는 것만 다시 보게 해줘. 색은 주황색으로."
🗣️ ④ 배포
"인터넷에 올려서 학생들이 링크로 쓰게 GitHub·Vercel 배포를 도와줘."

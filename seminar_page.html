02 — CURRICULUM

당일 만드는 것들

수업이 끝나면 실제로 작동하는 시스템이 손에 남습니다.

  • 학생 DB + 교재 DB 구축
    학생 명단·연락처·재원상태, 사용 교재·유닛 목록을 데이터베이스로 정리. 필터로 재원생만, 정렬로 요일별 — 원하는 대로 꺼내봅니다.
    실습
  • DB끼리 연동하기 — 관계형 + 롤업
    학생 DB와 교재 DB를 관계형으로 연결. 수업 기록에서 학생 이름을 선택하면 연락처가 자동으로 따라오는 그 마법을 직접 만듭니다.
    핵심 개념
  • 요일별 시간표 만들기
    월~금 수업 스케줄을 DB로 관리. 그룹화 기능으로 요일별 블록 보기, 선생님별 필터링까지.
    실습
  • 일일 리포트 DB + 학부모 문자 자동 완성
    수업 기록을 입력하면 학부모 문자가 자동으로 완성됩니다 — 복사해서 카톡에 붙여넣기만 하면 끝. 타이핑 없이 10초면 발송.
    하이라이트
  • 교재비 관리 DB + 학부모 문자 보내기
    교재 구입 내역·미납 현황을 DB로 관리. 수식으로 안내 문자를 자동 완성하는 방법까지.
    실습
  • Make로 일일 리포트 자동 SMS 발송
    노션에 수업 기록을 입력하는 순간 Make가 감지 → 솔라피로 학부모 문자 자동 발송. 당일 본인 폰으로 직접 테스트합니다.

    카카오 알림톡은 템플릿 검수에 1~3일이 걸려 당일 테스트가 어렵습니다. 이번 세미나에서는 SMS로 자동 발송을 경험하고, 세미나 2에서 카카오톡 알림톡으로 업그레이드합니다.

02 — SEMINAR 1 · CURRICULUM
세미나 1 · 노션 + Make

당일 만드는 것들

노션이 처음이어도 괜찮습니다. 기초 기능부터 차근차근, 수업이 끝나면 실제로 작동하는 학원 시스템이 손에 남습니다.

  • 노션 기초 · 데이터베이스 다루기
    페이지·블록 개념부터 표 만들기, 속성 추가, 그리고 같은 데이터를 표·보드·캘린더·갤러리로 바꿔 보는 뷰 전환. 필터·정렬·그룹화로 "재원생만", "요일별로", "선생님별로" 원하는 대로 꺼내봅니다.
    기초
  • 학생 DB + 교재 DB 구축
    학생 명단·연락처·재원상태, 사용 교재·유닛 목록을 데이터베이스로 정리. 학원 운영의 뼈대가 되는 핵심 DB를 직접 만듭니다.
    실습
  • DB끼리 연동하기 — 관계형 + 롤업
    학생 DB와 교재 DB를 관계형으로 연결. 수업 기록에서 학생 이름을 선택하면 연락처가 자동으로 따라오는 그 마법을 직접 만듭니다.
    핵심 개념
  • 출결·보강 관리 + 요일별 시간표
    하나의 DB를 여러 뷰로 — 오늘 보강, 일주일 뒤 보강(캘린더 뷰), 이번 달 결석을 한눈에. 월~금 시간표는 그룹화로 요일별 블록 보기까지.
    실습
  • 버튼 + 원장 스케줄 캘린더
    클릭 한 번으로 오늘 날짜 리포트·일정이 자동 추가되는 버튼 만들기. 상담·행정·수업 일정을 분류별 색으로 보는 원장 전용 캘린더로 하루를 한눈에 관리합니다.
    실습
  • 일일 리포트 DB + 학부모 문자 자동 완성
    수업 기록을 입력하면 학부모 문자가 자동으로 완성됩니다 — 복사해서 카톡에 붙여넣기만 하면 끝. 타이핑 없이 10초면 발송.
    하이라이트
  • 교재비 관리 DB + 학부모 문자 보내기
    교재 구입 내역·미납 현황을 DB로 관리. 수식으로 안내 문자를 자동 완성하는 방법까지.
    실습
  • Make로 일일 리포트 자동 SMS 발송
    노션에 수업 기록을 입력하면 Make가 감지 → 솔라피로 학부모 문자 자동 발송. 당일 본인 폰으로 직접 테스트합니다.

    카카오 알림톡은 템플릿 검수에 1~2영업일이 걸려 당일 테스트가 어렵습니다. 세미나 1에서는 SMS로 자동 발송을 경험하고, 세미나 2에서 알림톡 직접 발송으로 업그레이드합니다.
    자동화

온라인 ZOOM · 약 3시간 · 코드 자료실 제공

  • 🎯
    오늘의 등원생 자동 판단
    요일만 보면 오늘 누가 오는지 코드가 알아서 판단. 전체학생 DB에서 오늘 등원생 명단을 자동으로 뽑아옵니다.
  • 📝
    데일리리포트 자동 생성
    오늘 등원생별로 빈 리포트가 자동으로 만들어집니다. 선생님은 진도·태도만 채우면 끝.
  • 📋
    웹앱 출결현황판 + 터치 출결
    태블릿·폰으로 오늘 등원생 명단을 띄우고 터치 한 번으로 등원·결석·지각 체크. 결석은 출결보강 DB에 자동 기록됩니다.
    하이라이트
  • 💬
    카카오 알림톡 직접 발송
    Make 없이 Apps Script가 솔라피를 직접 호출해 알림톡 발송. 중간 다리 없이 코드에서 바로 학부모에게 전송합니다.
  • 🤖
    월말 평가서 자동 생성
    한 달 누적 수업 기록 → 평가서 자동 작성 → 학부모 열람 링크 생성. 출석률·태도·단어통과율이 도넛차트로 정리됩니다.
    하이라이트
⚠️ 세미나 2 수강 전, 꼭 미리 준비하세요
